Découverte du langage PHP

Qu'est-ce que PHP ?

PHP (PHP: Hypertext Preprocessor) est un langage de programmation côté serveur, spécialement conçu pour le développement web.

Concepts de base

  • Variables et types de données
  • Structures de contrôle
  • Fonctions
  • Tableaux

Projets réalisés

  • Page de contact
  • Formulaire de connexion
  • Gestion de sessions
  • Validation de données

Exemple de code PHP

<?php
    $nom = "Waël";
    $age = 18;
    
    echo "Bonjour, je m'appelle " . $nom;
    echo " et j'ai " . $age . " ans.";
?>
